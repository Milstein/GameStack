SceneCrossfader Class

A special type of scene that manages other scenes and adds the ability to cross-fade between the two.

Syntax

public class SceneCrossfader : Scene, IHandler<Resize>, IHandler<Start>, IUpdater, IDisposable

Remarks

This class uses a GameStack.Graphics.FrameBuffer to store the rendered contents of one scene while transitioning to the next. By blending the two, a smooth fading effect is achieved.
