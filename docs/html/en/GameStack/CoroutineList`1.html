CoroutineList<T> Generic Class

Manages a list of coroutines, stepping them forward on each update. Coroutines can be added or stopped at any time.

Syntax

public class CoroutineList<T> : IUpdater
where T : class

Type Parameters

T
The type of object that will be passed in on each update.

Remarks

Commonly used with the GameStack.FrameArgs class.
