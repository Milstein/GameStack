Apply Method

Apply the lighting rig to the currently active material/shader.

Syntax

Parameters

world
World transform applied to lighting rig.

Remarks

Usually called by drawable classes and not directly by apps. If no material is in scope, an exception will be thrown. If the material does not support lighting, the behavior of this method is undefined.
