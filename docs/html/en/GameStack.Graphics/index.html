GameStack

GameStack.Graphics Namespace

Namespace

Type Description
Animation Manages animations provided by model assets.
AnimationBehavior Indicates how an animation should behave when animating a time outside of its defined timeline.
AnimationChannel Manages a set of transform key frames to be applied to a mesh node when playing an animation.
Atlas Manages a collection of sprites combined together into a single texture (a spritesheet).
Batch Combines multiple draw operations with one material into a single draw operation.
BitmapChar Holds information about a single bitmap character glyph.
BitmapFont Manages a bitmap font asset.
BlendMode Indicates which blend mode to use with GameStack.Graphics.MeshMaterial
BlurDirection Indicates which direction (horizontal or vertical) an instance of a blur shader should use.
BlurMaterial A material applies a blur affect to anything rendered using it.
BlurShader Custom shader providing horizontal and vertical Gaussian blur.
Bone Manages a single bone instance, used with model animation.
Camera Manages projection and view transforms.
Camera2D A utility class that adds convenience functions to GameStack.Graphics.Camera specific to 2D scenarios.
Camera2DOrigin Indicates where the (0,0) coordinate is considered to be with repsect to the camera.
DirectionalLight Simulates a light at an infinite distance shining rays in parallel in a specified direction.
Drawable2D An abstract base class for any simple 2D drawable object such as a sprite or quad.
FrameBuffer Provides the ability to render to a surface other than the viewport, allowing apps to use the output of the rendering to apply further effects.
FrameSequence<T> Abstract base class for classes providing functionality related to sequences of frames.
FreeTypeException Thrown whenever there is a problem loading or initializing a font with FreeType.
FreeTypeFontFace Manages a vector font face for use with the drawing API.
HorizontalAlignment Indicates horizontal alignment for UI elements.
IBatchable When implemented by an object, indicates that the object is capable of being used with the GameStack.Graphics.Batch class to batch draw calls.
IndexBuffer Manages a native index buffer containing a set of indices into a GameStack.Graphics.VertexBuffer.
Light Abstract base class for different types of light.
Lighting Aggregates one or more lights into a lighting rig to be used with drawing operations.
LightType Indicates which type of light a light source provides.
Material A base class for all materials.
Mesh Manages a single mesh within a GameStack.Graphics.Model.
MeshMaterial A material to be used with the default mesh shader.
MeshShader Default mesh shader implementing a standard Phong model.
MeshShaderSettings Controls the limits of a mesh shader instance.
MeshTextureSlot Maps a mesh to one or more textures.
Model Manages a model asset.
Node A single node in a model's transform hierarchy.
PointLight Simulates a light that sends rays in all directions from a specific point.
PositionKey Represents a single position key in an animation channel.
Quad A convenience type that can render a simple quad using a specified material.
RotationKey Represents a single rotation key in an animation channel.
ScalingKey Represents a single scaling key in an animation channel.
ScopedObject An abstract base class for objects that can be added to and removed from the drawing scope.
Shader A base class for custom shaders, providing basic shader compilation, linking and reflection facilities.
ShaderException Thrown when there is a problem compiling, linking, or using a shader.
ShadingMode Indicates the desired shading model for a mesh.
SlicedSprite Extends the GameStack.Graphics.Sprite class and provides the ability to resize a sprite without distorting the edges or border.
SolidColorShader A convenience shader for drawing solid colors (e.g., for debugging).
Sprite Manages a single 2D image, often part of a spritesheet.
SpriteMaterial The default material used with GameStack.Graphics.Sprite and GameStack.Graphics.Atlas.
SpriteSequence Utility class for drawing a sequence of sprite images at a specified frame rate.
SpriteShader Default shader for sprites and 2D objects.
TextBlock A primitive drawable bitmap text object containing one or more lines of text.
Texture Manages a texture, or an area of memory containing an image or map used in rendering.
TextureFilter Indicates what kind of min or mag filtering a texture uses.
TextureOperation Indicates which texture operation to use when combining texture layers in a mesh material.
TextureSettings Options container for per-texture rendering options.
TextureSlot Maps one or more textures to a mesh.
TextureWrap Controls how textures are wrapped; specifically, how textured are rendered with a U or V coordinate outside the 0..1 range.
VertexBuffer Manages a collection of vertices in a buffer.
VertexElement Represents a single element within a GameStack.Graphics.VertexFormat.
VertexFormat A vertex format specifies which attributes are passed into the shader for each vertex.
VertexWeight Used with model animations to specify a weight for a vertex or group of vertices.
