Batch Class

Combines multiple draw operations with one material into a single draw operation.

Syntax

public class Batch : ScopedObject, IDisposable

Remarks

When an instance of this class is placed in scope (by calling Begin), draw calls on supported objects (those implementing GameStack.Graphics.IBatchable) will be batched into a single draw call per material. The actual draw takes place when the batch leaves scope.
