VertexBuffer Class

Manages a collection of vertices in a buffer.

Syntax

public class VertexBuffer : ScopedObject

Remarks

Collections of vertices can belong to simple 2D geometry like sprites or quads, and 3D meshes. This class is typically used in concert with GameStack.Graphics.IndexBuffer. By using a separate index buffer, vertices can be re-used any number of times in the mesh whenever the attributes are identical. In order to draw with an index buffer, an instance of this class must be in scope. This class is not normally created directly, but instead via one of the asset loading classes like GameStack.Graphics.Model.
