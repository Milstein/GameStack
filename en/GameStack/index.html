GameStack Namespace

Namespace

Type Description
Assets Static class for accessing assets on the current platform.
CairoEx Convenience extension methods for working with Cairo.
Controller<T> Controls a property or variable by transitioning it from value to value over time.
ConversionEx Static utility methods for common conversions.
CoroutineList<T> Manages a list of coroutines, stepping them forward on each update. Coroutines can be added or stopped at any time.
EventBase Abstract base class for all GameStack events.
FrameArgs Contains information for a single frame.
Gesture Abstract base class for all gesture events.
GestureState Indicates the phase that a gesture is currently in.
GestureType Indicates the type of the current gesture.
ICoroutine Coroutines are controlled through this interface.
IGameView A cross-platform interface for platform-specific game view implementations.
IGestureInput Implemented by all UI views that which to receive gesture events within their frame.
IHandler<T> Any class can implement this interface to receive the specified type of event.
IStreamingSound Generic interface for a streaming sound asset.
IUpdater Implemented by any object that wants to receive an update each frame.
IWaitFor Implemented by any type that can be yielded from a coroutine.
IWaitFor<T> Implemented by any type that can be yielded from a coroutine.
KeyEvent An object of this type is added to the queue whenever a keyboard event occurs.
KeyModifier Indicates which modifier keys are pressed during key events.
KeyState Indicates the current state of a key associated with a GameStack.KeyEvent.
LowMemory An event type that signals the need to release unneeded resources.
Mathf Static utility methods for manipulating single-precision floating point values.
OpenALException Thrown when an OpenAL operation has failed.
PanGesture Instances of this class are added to the event queue when pan gestures have been detected.
Pause An instance of this type is added to the event queue when the scene has been paused.
Ray A utility class representing a ray.
Resize An instance of this type is added to the event queue when the viewport has resized.
Resume An instance of this type is added to the event queue when the scene has been resumed from a paused state.
Scene The scene is responsible for updating its own objects, managing its own resources, and controlling any number of entities or views. Most apps will have types derived from Scene.
SceneCrossfader A special type of scene that manages other scenes and adds the ability to cross-fade between the two.
SoundChannel A single sound channel through which one sound at a time may be played.
SoundEffect An individual sound effect asset, commonly a short sound snippet that will be buffered in memory.
Start An instance of this type is added to the event queue when the scene is clear to begin running.
StreamingOpusFile A sound asset capable of streaming rather than buffering entirely into memory.
StreamingSoundChannel A sound channel that can be used to play sounds streamed from asset resources.
SwipeDirection Indicates a swipe gesture's direction.
SwipeGesture An instance of this type is added to the event queue whenever a swipe gesture is detected.
TapGesture Instances of this type are added to the event queue whenever a tap gesture is detected.
ThreadContext Provides static access to the state of the current thread.
Touch Instances of this type will be added to the event queue whenever a touch event occurs.
TouchState Indicates the current phase of an individual touch.
Transition<T> Transitions a property from one value to another over time.
Tween Static class providing common easing functions.
TweenFunc<T> Functions matching this delegate can be used with GameStack.Controller<T> and GameStack.Transition<T>.
WaitFor Any class deriving from this class can be yielded from a coroutine.
WaitFor<T> Any class deriving from this class can be yielded from a coroutine.
WaitForAny<T> Aggregates several GameStack.IWaitFor objects to suspend execution until any have completed.
WaitForCoroutine When yielded from a coroutine, waits for a specified coroutine to complete before continuing execution.
WaitForTime When yielded, suspends execution until the specified period of time has passed.
