IHandler<T> Generic Interface

Any class can implement this interface to receive the specified type of event.

Syntax

public interface IHandler<T>
where T : GameStack.EventBase

Type Parameters

T
The type of event to be handled.

Remarks

Scenes can implement this interface, as well as objects that are added to a scene collection. When added, any object that has an interface of any specific type will handle events of that type that come through the queue.
