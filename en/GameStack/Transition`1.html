Transition Constructor

Start a transition immediately.

Syntax

public Transition (T from, T to, float duration, TweenFunc<T> easing, Action<T> callback, Action done)

Parameters

from
The starting value.
to
The ending value.
duration
The transition duration in seconds.
easing
The easing function to use.
callback
Called to set the controlled object's property.
done
Called when the transition has finished.

Remarks

In order to continue running, the transition must either be added to the scene collection by calling Scene.Add(object) or by pumping the transition manually on update.
