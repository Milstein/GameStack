LoadFrame Method

Mark the next frame as a load frame.

Syntax

public void LoadFrame ()

Remarks

Load frames are frames that are especially long due to heavy operations such as loading many assets. The purpose of distinguishing load frames is so that transitions and animations don't skip ahead during heavy operations. The event pump will not try to "catch up," but will instead proceed to the next frame as if a usual amount of time has passed.

Update Event

Fires when the current scene should update.

Syntax

public event EventHandler<FrameArgs> Update

Remarks

GameStack does NOT guarantee that there is a valid graphics or audio context when this event fires. To perform rendering operations, use the IGameView.Render event. This event is for logical updates.
