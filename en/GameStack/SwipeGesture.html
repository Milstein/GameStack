SwipeGesture Class

An instance of this type is added to the event queue whenever a swipe gesture is detected.

Syntax

public class SwipeGesture : Gesture

Remarks

Only supported on platforms with gesture support (currently mobile). For events of this type to be generated, the IGameView.EnableGesture(GestureType)method must be called to enable this gesture type.
