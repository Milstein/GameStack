Touch Class

Instances of this type will be added to the event queue whenever a touch event occurs.

Syntax

public class Touch : EventBase

Remarks

Touch events include things like finger down, repeated movements, and finger up. For higher level gesture support (where available) see classes derived from GameStack.Gesture. On desktop platforms, mouse events will act like touch events (with left click representing a finger down action).
