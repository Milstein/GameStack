GameStack.Content Namespace

Namespace

Types in this namespace are generally only used by classes that load and control assets. You can use these tools to build custom multi-file asset types based on the standard .tar file format.

Type Description
BinaryEx Static extension methods for reading and writing binary data.
ContentException Thrown whenever an asset fails to load or could not be found.
ITarHeader Metadata and file access for a single entry in a .tar file.
PngLoader Static facilities to load PNG files.
TarException Thrown when an error has occurred reading a multi-file TAR asset.
TarReader Provides basic .tar file reading facilities for asset loaders.
