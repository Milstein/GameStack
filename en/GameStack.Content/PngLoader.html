PngLoader Class

Static facilities to load PNG files.

Syntax

public static class PngLoader

Remarks

This facility assumes that PNG files have already been premultiplied by the aset pipeline. This class is implemented for each platform separately. Generally, this class is used by asset loaders. For normal image drawing, use an GameStack.Graphics.Atlas or GameStack.Graphics.Quad and GameStack.Graphics.Texture.
