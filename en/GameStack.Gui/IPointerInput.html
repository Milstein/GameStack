IPointerInput Interface

Implemented by any UI controls that wish to receive generic pointer events.

Syntax

public interface IPointerInput

Remarks

Pointers are abstract and take a different form on different platforms. On mobile, a pointer refers to a single tracked touch. On a desktop, the pointer is simply the mouse cursor. Most UI interfaces involve a single pointer.
