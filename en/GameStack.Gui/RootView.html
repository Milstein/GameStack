RootView Class

Manages a UI hierarchy and processes events handled by controls within the hierarchy.

Syntax

public class RootView : View, GameStack.IHandler<GameStack.Gesture>, GameStack.IHandler<GameStack.Resize>, GameStack.IHandler<GameStack.Touch>, GameStack.IUpdater

Remarks

An instance of this class must sit at the root of any UI hierarchy. It represents the entire hierarchy to your scene. Add the root to the scene collection by calling GameStack.Scene.Add(object) to pass events into the hierarchy.
