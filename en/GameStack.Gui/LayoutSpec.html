LayoutSpec Class

A layout specification is a collection of lambda functions that control the behavior of a view's layout with respect to its parent.

Syntax

public class LayoutSpec

Remarks

Since layout specs use lambdas, they can be continually evaluated whenever a layout refresh is required. This provides an extremely flexible layout system capable of stacking, tiling, grids, and so on.
