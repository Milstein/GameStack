ImageView Class

A simple view for displaying a single image.

Syntax

public class ImageView : View

Remarks

If a GameStack.Graphics.SlicedSprite is provided, the sprite is automtically resized to fit the view frame. Since this operation mutates the sprite's geometry, consider calling GameStack.Graphics.SlicedSprite.Clone() to create a copy first. Sliced sprites are resized regardless of the ImageView.Fill setting.
