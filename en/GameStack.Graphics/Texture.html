Texture Class

Manages a texture, or an area of memory containing an image or map used in rendering.

Syntax

public class Texture : IDisposable

Remarks

GameStack standardizes on .png files for texture images. The pipeline tools will automatically pre-multiply both stand-alone images and images that are part of a spritesheet.
