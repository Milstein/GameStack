SlicedSprite Class

Extends the GameStack.Graphics.Sprite class and provides the ability to resize a sprite without distorting the edges or border.

Syntax

public class SlicedSprite : Sprite

Remarks

Instances of this class are typically retrieved via Atlas.GetSprite``1(string).

A sliced sprite is broken into 9 quads: one for each edge, one for each corner, and one for the middle (think of a tic-tac-toe board). This allows each portion to be scaled differently. Corners are never scaled. Edges can be scaled along their axis, but not long their width (or they can be tiled along their axis). The center is scaled in either direction, or it can tile in either direction (or both).
