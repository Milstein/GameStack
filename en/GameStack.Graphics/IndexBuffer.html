IndexBuffer Class

Manages a native index buffer containing a set of indices into a GameStack.Graphics.VertexBuffer.

Syntax

public class IndexBuffer : IDisposable

Remarks

Index buffers are used to optimize memory by eliminating redundant vertex definitions. Each vertex can be referenced multiple times in a mesh's geometry by index.
