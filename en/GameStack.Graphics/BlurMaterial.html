BlurMaterial Class

A material applies a blur affect to anything rendered using it.

Syntax

public class BlurMaterial : SpriteMaterial

Remarks

A horizontal or vertical blur may be applied, but not both at the same time. To blur in both directions, use a GameStack.Graphics.FrameBuffer object for multi-phase drawing.

BlurMaterial Constructor

Syntax

public BlurMaterial (Shader shader, Texture texture, float texelSize)

Parameters

shader
If provided, always use the custom shader instead of the default blur shader.
texture
When blurring a textured object, a texture must be specified.
texelSize
The blur shader needs to know the size of each texel so it can calculate blur radius propertly. This is normally equal to 1 / texture width for horizontal blur and 1 / texture height for vertical blur.
