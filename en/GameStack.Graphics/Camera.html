Camera Class

Manages projection and view transforms.

Syntax

public class Camera : ScopedObject

Remarks

For 2D requirements, consider using the GameStack.Graphics.Camera2D convenience class. Whenever a camera is in scope (by calling Begin) the camera's transforms will be applied to drawing operations. Most drawing operations require a camera and will throw an exception if none is in scope.
