IBatchable Interface

When implemented by an object, indicates that the object is capable of being used with the GameStack.Graphics.Batch class to batch draw calls.

Syntax

public interface IBatchable

Remarks

Any object implementing IBatchable has an additional responsibility: when the object is asked to draw itself, it must check if there is a GameStack.Graphics.Batch is in scope using the ScopedObject.Find<T> method. If there is, it must call Batch.Draw(IBatchable) instead of drawing itself normally.
