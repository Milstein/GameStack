FrameSequence<T> Generic Class

Abstract base class for classes providing functionality related to sequences of frames.

Syntax

public class FrameSequence<T> : GameStack.IUpdater

Type Parameters

T
The type of object to be sequenced.

Remarks

Add this object to your scene collection to automatically update it each frame. Otherwise, pump it manually in your update method.
