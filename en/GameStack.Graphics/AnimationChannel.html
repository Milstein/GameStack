AnimationChannel Class

Manages a set of transform key frames to be applied to a mesh node when playing an animation.

Syntax

public class AnimationChannel

Remarks

One animation channel can work in concert with others to simultaneously animate different parts of the model. This class is normally instantiated by asset loaders, and can be retrieved from instances of GameStack.Graphics.Animation.
