Start Method

Begins playing an animation.

Syntax

public void Start (double startTime)

Parameters

startTime
The point in the animation at which to start, or 0 for the beginning.

Remarks

Once called, the animation begins applying transforms to the mesh nodes each time it is updated. If your animation is not updating, ensure that this object has been added to the scene collection by calling GameStack.Scene.Add(object) or that you are manually pumping the animation in your update method.
